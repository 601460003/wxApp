###### 如何使用
###### 
 


```
git clone jianxin@192.168.1.252:newlife/newlife_yun_charing.git
cd newlife_yun_charing
npm install --registry=https://registry.npm.taobao.org
npm start 启动监听文件改动 

```

#### 小程序操作
1. 小程序工具更新最新 
2. 导入newlife_yun_charin/dist/wx 
3. 开启 es6->es5
